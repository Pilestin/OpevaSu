# OpevaSu Mobile Workspace

Bu klasor iki parcadan olusur:
- `app/` -> Expo React Native uygulamasi
- `backend/` -> Node.js + Express API

## Kurulum

```bash
cd mobile
npm install
npm run install:all
```

## Calistirma

Tek komutla:

```bash
npm run dev
```

Ayrica ayri ayri:

```bash
npm run dev:backend
npm run dev:app
```
